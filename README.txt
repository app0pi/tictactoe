A2 / TicTacToe

The program is a two-player tic-tac-toe game with settings for dark mode or X going first with a database.

Instructions:
-Press New Game to begin a game
-Two players take turns playing by pressing squares
-All completed games are logged in the database

Compiling Code:
-ignore the Kotlin compiler warning, if it persists, press build and run again

Bugs:
-this is not really a bug but since the toast doesn't disappear immediately it may lag behind actions
